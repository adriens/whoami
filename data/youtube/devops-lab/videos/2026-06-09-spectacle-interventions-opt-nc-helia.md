SDK helia-etat-reseaux (PyPI) — scraping des maintenances programmées réseau OPT-NC/HELIA (helia.nc/etat-du-reseau).
Enrichissement avec les référentiels open data NC : communes (datavenc), provinces et population légale.
Data repo GitHub accumule les interventions actives et archivées.
Application web Astro déployée sur GitHub Pages : carte interactive géolocalisée, vue Kanban, analytics, RSS et calendrier (.ics) par commune.
"Tour des maintenances" : animation data-art spectacle des interventions OPT-NC sur fond cartographique.
Rétrospective sur la saga temps d'attente OPT-NC (Unity 3D, LED matrix, visualisations reprises dans le plan stratégique).
Lien explicite avec la méthode 5S data-driven.
