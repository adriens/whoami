## 💭 First thoughts

Since I discovered `ollama` many months ago, day after day the feeling that the number of models (and tags) was going ever bigger...somewhat _out of control_.

Lastly, news were so rich and models were landing at an ever increasing pace:

{% twitter 1780598146470379880 %}

In the mean time, I was not the only one to find _the were a lot of models out there_ : 

{% twitter 1781431672262525094 %}

👉 So I started to build a place to **put all `ollama` library data altogether so anyone can use datascience on top of it for any purpose.**

This post is all about this.

## 🎯 Goals

1. **Get** data together
2. **Link** data together
3. **Schedule data delivery** as Kaggle Dataset
4. **Make some datastorytelling** around MT-Bench and [`duckdb` `FTS` (Full-Text Search Extension)](https://duckdb.org/docs/extensions/full_text_search.html) and the automation opportunities we have
