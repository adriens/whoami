## 💭 About

Imagine the following data:

- **🎯 A set of strategic goals** (and related KPIs) described from APIs
- **🗂️ A collection of datasets**, delivered as `csv` (and metadatas)

... now imagine you have **a set of keys that make it possible to link them.**

👉 This pattern is a very common one, both **for profit or non-profit organizations as any organization's aim is to achieve strategic goals... and if it's data-driven...based on data.**

## 🎯 What we'll do

Well, this is what was done during my `#NODES23` session, and today I'll show the benefits of this approach on a very concrete case : **discover everything we can know about women in a country, from a strategic data-driven perspective, interactively and within less than 10'.**

In our case, we'll use : 

- **🎯 Strategic goals** from [UN SDGs](https://sdgs.un.org/goals) and Open Data from [Pacific Data Hub](https://pacificdata.org/) 
- **🗂️ Datasets** from [`data.gouv.nc`](https://data.gouv.nc/) Open Data
